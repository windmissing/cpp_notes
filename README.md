# 相比于其它语言，C++有什么优势

1. 相比于高级语言，C++更贴近硬件  
C++能使用原生和指令和类型，性能非常高  
C++可以方便地使用新的硬件，例如GPU、FPGA

2. 相比于低级语言，C++具有零开销抽象  
如果没有在代码中使用抽象，就不会有额外的开销  
如果在代码中使用了抽象，会有额外的开销，但这个开销非常低

# C、C++、现代C++

1. 从功能上，C++是以的超集  
2. C++比以更严格，也更安全。例如const的使用、指针的转换、枚举类型的转换  
3. 现代C++和惯用法与C大相径庭
