下面程序的输出结果为（）

```C
unsigned int a=1;  
cout<<a*-2<<endl;  
```

A、-4      B、4       C、4294967294         D、4294967295

答：C

考查的是unsigned int和int在一起混合运算，int转化为unsigned int
-2的补码就是2^32-2，即是4294967294 ，乘以1的结果还是这个数字。
